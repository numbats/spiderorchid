# staff_ids

This dataset contains the mappings between researcher names and their
respective ORCID and Google Scholar IDs. It is useful for identifying
and linking academic profiles across different platforms.

## Usage

``` r
staff_ids
```

## Format

A data frame with 4 variables:

- first_name:

  `character`. The first name of the individual.

- last_name:

  `character`. The last name of the individual.

- orcid_id:

  `character`. The ORCID identifier.

- scholar_id:

  `character`. The Google Scholar user ID.
